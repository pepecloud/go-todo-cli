# Go ToDo CLI

Мой первый учебный пет-проект на языке Go. Консольное приложение для управления задачами.

## Возможности
- Добавление задач
- Просмотр списка
- Удаление задач
- Отметка о выполнении
- История событий (логов)

## Как запустить
1. Склонируйте репозиторий
2. Запустите через `go run main.go`

## Стек
- Golang
- Библиотека `k0kubun/pp` для красивого вывода
